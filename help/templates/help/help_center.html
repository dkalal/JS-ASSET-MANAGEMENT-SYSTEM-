Getting Started

System Overview

The Enterprise Asset Management System helps you track, manage, and maintain your organization's assets efficiently.

Key Features:
  • Asset Tracking: Complete lifecycle management
  • QR Code Scanning: Quick asset identification
  • Reporting: Comprehensive analytics and reports
  • User Management: Role-based access control
  • Audit Trail: Complete activity logging
User Roles:
Admin
Full system access, user management, settings
Manager
Asset management, reports, team oversight
User
Asset viewing, scanning, basic operations

Asset Management

Adding New Assets
  1. Navigate to Assets → Add Asset
  2. Fill in required information (Name, Category, Location)
  3. Add optional details (Serial Number, Purchase Date, Value)
  4. Upload photos if available
  5. Click Save Asset
Asset Categories

Organize assets into categories for better management:

  • IT Equipment: Computers, servers, networking
  • Furniture: Desks, chairs, office furniture
  • Vehicles: Company cars, trucks, equipment
  • Machinery: Manufacturing equipment, tools
Bulk Import

Import multiple assets at once using Excel/CSV files:

  1. Download the import template
  2. Fill in asset information
  3. Upload the completed file
  4. Review and confirm imports

Scanning & Tracking

QR Code Scanning

Each asset has a unique QR code for quick identification:

  1. Navigate to Scan page
  2. Allow camera access
  3. Point camera at QR code
  4. View asset details instantly
Asset Tracking

Track asset location and status changes:

  • Location updates
  • Status changes (Active, Maintenance, Retired)
  • Assignment to users
  • Maintenance schedules
